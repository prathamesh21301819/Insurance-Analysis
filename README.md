📊 Power BI Dashboard Project
🚀 Overview

This repository contains a Power BI (.pbix) dashboard designed to transform raw data into meaningful insights using interactive visualizations. The dashboard helps users analyze trends, monitor KPIs, and make informed decisions efficiently.

📌 Features

📈 Interactive and dynamic visualizations

🎯 KPI indicators for performance tracking

🔍 Advanced filtering and slicing options

⚡ Clean and intuitive dashboard layout

🔄 Data transformation using Power Query

🧮 Custom calculations using DAX

🛠️ Tech Stack

Power BI Desktop

Power Query (ETL operations)

DAX (Data Analysis Expressions)
