📊 Insurance Analytics Dashboard – Power BI
🚀 Overview

This project presents an interactive Insurance Analytics Dashboard built using Power BI. It provides deep insights into insurance data such as premium amounts, claims, coverage, and customer distribution.

The dashboard is designed to help stakeholders analyze policy performance, claim trends, and customer demographics for better decision-making.

🎯 Key Insights

💰 Total Premium Amount: 5.98M

🛡️ Total Coverage Amount: 600.55M

📉 Total Claim Amount: 16.91M

👥 Gender distribution: ~5000 Male & Female customers

📊 Claims categorized as Rejected, Settled, and Pending

📌 Features
📈 Interactive Dashboard

Dynamic filters for:

Policy Number

Claim Number

Customer ID

📊 Data Visualizations

Premium Amount by Policy Type (Travel, Health, Auto, Life, Home)

Claim Amount by Age Group (Adult, Elder, Young Adult)

Claim Status Distribution (Rejected, Settled, Pending)

Active vs Inactive Policies

📋 Detailed Data Table

Customer-level data including:

Age, Gender

Policy Type

Premium & Coverage Amount

Claim Status & Dates

🛠️ Tools & Technologies

Power BI Desktop (.pbix)

Power Query – Data cleaning & transformation

DAX – Measures and calculated fields


KPI cards for Premium, Coverage, and Claims

Policy type and claim analysis visuals

🔹 Detailed Data View

Tabular dataset with complete customer and policy information
